# Copy & Language Guidelines

**Tone**: Ritual meets Moon Juice. You know exactly what's in it, but it feels desirable, not like a supplement label.

---

## Badge system

Two visual styles for ingredient badges:

### Vitamin code badges (compact, clinical credibility)
- Appearance: solid background, sharp corners (border-radius: 4px), bold weight
- Content: short codes — C, D, B12, B3, B5, B6, E, B1
- Purpose: signal "real vitamins are in this" without clinical overload

### Botanical / fruit badges (warmer, organic)
- Appearance: transparent background, rounded (border-radius: 8px), italic, softer border
- Content: full ingredient names — Ginger, Mint, Chamomile, Hibiscus, Pomegranate, Peach, Açaí, Blackcurrant, Berry
- Purpose: signal "natural, recognisable ingredients" — the lifestyle layer

### What never appears in badges
- Dosages (80mg, 5μg)
- NRV percentages
- Ingredient weights
- Clinical terminology

## Boost card copy

**Structure**: Aspirational lifestyle name → benefit headline → ingredient badge

**Pattern**:
```
[Lifestyle name]                          ← hero, large
[Benefit that pairs feeling + function]   ← subtitle
[Badge: ingredient code]                  ← small tag
```

**Example**:
```
Glow & Move Freely
Supports skin elasticity and joint recovery
[Collagen]
```

**Rules**:
- Name is aspirational, not clinical ("Glow & Move Freely" not "Joint & Skin Support")
- Benefit copy names the outcome AND the mechanism
- Badge is the ingredient, not the benefit

---

## Flavour card copy

**Structure**: Flavour name (hero) → function name (secondary) → vitamin + botanical badges (tertiary)

**Pattern**:
```
[Flavour name]              ← hero, large
[Function name]             ← secondary, with colour dot
[C] [B5] [B6] [Chamomile]  ← grouped badges, compact
[Zero sugar]                ← tag, smallest
```

**Example**:
```
Mango Guava
● Unwind
[C] [B5] [B6] [B12] [Chamomile]
Zero sugar
```

**Rules**:
- Flavour name is ALWAYS the hero — it's a taste choice first
- Function name is a clear label, not a sentence
- Badges show what's inside without explaining what each does (that's what the benefit cards in the sidebar are for)
- Never show dosages on the flavour card

## Glass visualization labels

Labels inside glass layers name the ingredient category, not quantities.

**Pattern**: "[Vitamin code] · [Vitamin code] · [Botanical]"

**Examples**:
- `Vitamin C · D · Ginger`
- `B-Complex · C · Chamomile`
- `B-Complex · E · Hibiscus`
- `Collagen`
- `Electrolytes`
- `Caffeine`

**Rules**:
- Group B-vitamins as "B-Complex" when there are 3+ (B5, B6, B12)
- Spell out individual B-vitamins when there are only 1-2 (B1, B3)
- Botanical names included when present
- NEVER show quantities in glass labels

## Dispensing screen reveals

**Structure**: staggered 600ms intervals

**Phase 1 — Boost reveals** (shown first):
```
[+] [Dispensing benefit sentence]
    [Badge: ingredient]
```
Example: `[+] Rebuilding your muscles with Protein [Protein]`

**Phase 2 — Flavour vitamin + botanical reveals**:
Header: "Plus, you're also getting — [Function name] vitamins & botanicals"
Summary: `[Single sentence: dispenseBenefit]`
Then individual items:
```
[Badge] [Benefit sentence with vitamin name]
```
Example: `[C] Protects your body while you relax with Vitamin C`
Botanical example: `[Chamomile] Chamomile to calm your nervous system naturally` *(italic)*

**Rules**:
- Boosts ALWAYS revealed before flavour vitamins
- "Plus, you're also getting" header only appears when boosts were selected
- If 0 boosts, flavour vitamins show directly as "Your [Function] vitamins & botanicals"
- Botanical entries render in italic

## Screensaver copy

**Emotional slides**: Two-part typographic treatment.
```
[Muted prefix — small]     "Your daily dose of"
[Bold keyword — large]     "focus."
[Subtitle — medium]        "Vitamin D & natural caffeine, in a drink you'll actually crave."
```

**Rules**:
- Prefix is warm, not clinical
- Keyword is a single aspirational word, lowercase, with period
- Subtitle names the specific vitamins/ingredients — function-aware, not vague
- Never purely emotional ("Feel amazing!") — always grounds in what's actually in the drink
- Never clinical ("Contains 80mg Vitamin C per serving")

**Live data slides**: Machine stats + trending info.
```
"284 drinks crafted here today"
"Most popular: Protein boost + Lemon Immunity"
```

## Words to avoid

| Don't say | Say instead |
|-----------|------------|
| "Delivers X mg" | "Fights fatigue with Vitamin C" |
| "Contains" (as benefit) | Name the outcome + ingredient |
| "Supplement" | "Drink" or "daily ritual" |
| "Dosage" / "Serving" | Never reference quantities |
| "NRV" / "RDA" | Never reference |
| "Clinically proven" | Not claimable at current formulation |
| "Immunity boost" (as a generic) | Be specific: "Vitamin C, D & ginger" |
